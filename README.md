### Mastermind-game
Simple project of writing a mastermind game using TDD

### Build

[![Build Status](https://travis-ci.org/Arakis14/Mastermind-game)]

### Compilation

```bash
mkdir build
cd build
cmake ..
make -j
```

### Tests 

```bash
build/Mastermind-gtest
```

### Run

```bash
build/Mastermind
```
